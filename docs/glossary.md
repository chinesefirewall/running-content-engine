# Glossary

## AI story engine

The part of the system that converts notes and metadata into titles, captions, hooks, overlay text, and story ideas.

## Content package

A set of platform-specific assets generated from one recording session.

Examples:

- YouTube title
- YouTube description
- Instagram caption
- TikTok hook
- Facebook post
- thumbnail ideas
- overlay text

## Daily folder

The local workspace for one recording day.

Example:

```text
content/2026/2026-07-04/
```

## Editing workflow

The process of turning clips and content assets into final videos.

This may involve:

- Insta360 App
- Insta360 Studio
- CapCut
- ChatGPT
- future tools

## Metadata

Structured information about a run or recording session.

Examples:

- distance
- pace
- heart rate
- weather
- shoe used
- mood
- lesson learned

## Overlay text

Short text displayed on video clips.

Example:

```text
18.2 km easy run
Avg HR: 146 bpm
Strong finish after 12 km
```

## Platform output

A final or semi-final content item prepared for a specific platform.

Examples:

- YouTube video
- YouTube Short
- Instagram Reel
- TikTok video
- Facebook Reel

## Processed footage

Footage that has been trimmed, stabilized, renamed, converted, or prepared for editing.

## Raw footage

Original video files from the camera or phone.

Raw footage should not be committed to the public repository.

## Story brief

A short written summary that explains the narrative angle of the day.

Example:

> Today's run started badly, but the final 5 km felt strong. The story is about showing up even when energy is low.
